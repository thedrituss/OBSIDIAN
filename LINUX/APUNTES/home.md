
`/home` (Your Personal Space)

- Where user accounts are stored.
- Primary location for user's files, documents, settings, and configs.
- Each user has their own subdirectory within `/home`.
	- user1 = `/home/user1`
- Key Directories:
	- __Documents__
	- __Downloads__
	- __Music__
	- __Pictures__
	- __Public__
	- __Videos__
- Permissions & Ownership:
	- __User ownership__
	- __Permissions__
	- __Personalization__
	- __Data Storage__
	- __Security__
